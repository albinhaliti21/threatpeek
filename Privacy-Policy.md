<h1>ThreatPeek Privacy Policy</h1>

ThreatPeek does not collect, store, or transmit personal user data.

The extension operates locally in the user's browser and processes Indicators of Compromise (IOCs) such as IP addresses, domains, and file hashes only when the user submits them for analysis.

When a lookup is performed, the submitted indicator may be sent directly to third-party threat intelligence services such as VirusTotal, AbuseIPDB, AlienVault OTX, or WHOIS providers in order to retrieve reputation and enrichment information. These requests occur only when initiated by the user.

ThreatPeek does not track browsing activity, collect analytics, or monitor user behavior.

API keys configured by the user for supported threat intelligence providers are stored locally using the browser's extension storage system so the extension can authenticate requests to those services. These keys are not transmitted to any servers operated by the extension developer.

Users should review the privacy policies of the third-party threat intelligence services they choose to use, as those services may process submitted indicators according to their own policies.
