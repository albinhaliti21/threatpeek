**ThreatPeek** is a fast IOC triage extension built for analysts, SOC teams, and incident responders.

It allows you to select or paste an IP address, domain, or file hash and instantly receive a consolidated verdict from multiple trusted threat intelligence providers, all in one place. ThreatPeek aggregates provider risk signals, influence, and contribution into a clear, transparent summary so you can make decisions faster without switching between multiple tools and tabs.

By combining enrichment from sources such as VirusTotal, **AbuseIPDB, AlienVault OTX, and WHOIS**, ThreatPeek transforms raw indicators into actionable triage results directly inside your browser.

<h2>Key Features</h2>

- Instant IOC analysis for IP addresses, domains, and file hashes
- Batch IOC scanning with sortable results and CSV export
- Multi-provider enrichment from VirusTotal, AbuseIPDB, AlienVault OTX, and WHOIS
- Transparent scoring model with provider-level risk breakdown, influence, and contribution
- History re-analysis to quickly re-check previously analyzed IOCs
- Provider toggles and BYOK API keys for flexible intelligence integration
- Optional AI Assist summaries for quick analyst context
- Light and dark theme support with a streamlined analyst-focused interface

<h2>Why Install ThreatPeek</h2>

- Analyze indicators faster without opening multiple threat intelligence platforms
- Make clearer decisions with transparent provider-based risk evidence
- Run batch IOC checks and sort results by risk level
- Export analysis results to CSV for reporting or further investigation
- Re-check previously analyzed indicators instantly from history
- Maintain full control with **Bring Your Own API Keys (BYOK)** and provider enable/disable settings
- Use optional AI summaries when additional context is helpful

<h2>Privacy and Control</h2>

ThreatPeek is designed with analyst workflows and data control in mind:

- Bring your own API keys for each provider
- Enable or disable intelligence sources based on your workflow
- Built for real-time SOC and incident response environments with minimal friction
